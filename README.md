# Neo Bank - Full Stack Banking Application

A modern full-stack banking application with Java Spring Boot backend and React frontend.

## Features

- User Authentication & Authorization (JWT)
- Account Management
- Money Transfers
- Transaction History
- Admin Dashboard
- Fraud Detection & Monitoring
- Personalized Financial Insights

## Tech Stack

**Backend:**
- Java Spring Boot
- MySQL Database
- Spring Security
- JWT Authentication

**Frontend:**
- React
- Tailwind CSS
- Axios

## Getting Started

### Backend Setup
```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### Frontend Setup
```bash
cd neo_bank-main
npm install
npm run dev
```

## Database Configuration

Update `backend/src/main/resources/application.properties` with your MySQL credentials.
